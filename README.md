# Algafood

Algafood é um projeto de API REST para gestão de restaurantes e pedidos. Desenvolvido para oferecer uma solução completa e escalável para o setor de alimentação, o Algafood visa simplificar e otimizar processos de gerenciamento de restaurantes e pedidos online.


## Funcionalidades

- Cadastro, atualização e remoção de restaurantes
- Cadastro, atualização e remoção de produtos
- Gestão de categorias de produtos
- Recebimento de pedidos
- Gerenciamento de status de pedidos
- Autenticação de usuários

## Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL
- Maven

## Como Executar
~~~
1. Clone este repositório:
git clone https://github.com/seu-usuario/algafood.git
~~~
~~~
2. Acesse o diretório do projeto:
cd algafood
~~~
~~~
3. Execute o comando Maven para compilar o projeto:
mvn clean install
~~~
```
4. Execute o projeto Spring Boot:
java -jar target/algafood.jar
```

## Documentação da API
